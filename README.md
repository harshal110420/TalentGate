# TalentGate
TalentGate is an enterprise-grade HR assessment and hiring platform designed to manage the complete recruitment lifecycle — from candidate onboarding and secure online examinations to automated evaluation, analytics, and shortlisting — all from a single unified dashboard.

TalentGate delivers:

Secure online exam execution with timed questions
Automated scoring with positive & negative marking logic
Candidate lifecycle management (Assigned → In Progress → Completed → Expired)
HR dashboard for results, rankings, and shortlisting
Token-based candidate authentication & exam security
Role-based user and admin access controls
REST API backend with Sequelize & MySQL
